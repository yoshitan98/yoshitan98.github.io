日本語→多言語辞書 JMdict を EPWING に変換する　2026年3月18日

　

毎日更新されている 日本語→多言語辞書 JMdict を EBStudio2 を使ってEPWING辞書に変換してみた。現在21万以上の語が登録されている。

上記リンク先 "The JMDict Project" の "the current version (Unicode UTF8) (Warning! This is a big file.)" をダウンローダーで落として7-Zip等を使って "JMdict" ファイルを抽出する。それをテキストエディターで見ると sd コマンドだけで EBStudio2 のソースとする html に変換できそうなのでバッチファイルを書いてみた。パイプをいくつもつなげているが sd の処理は高速なので問題なし。

"JMdict" を GNU sort を使って かな で事前にソートするバッチファイルも書いてみた。sort コマンドは Cygwin か MSYS2 をインストールして利用する必要があるが、GNU ではないが BusyBox for Windows の sort でも動作したので Cygwin や MSYS2 をインストールしていなければそれが最も簡単。

EBStudio2 はライセンスを購入してあるので全部の種類のインデックスを作成できるが、後方一致かなインデックスは全部の語がマッチせず使いものにならない。何か仕様のつもりかも知れないがバグと言っていいだろう。

同じ作者の定番ソフト EBWin4 で動作確認したが非ASCIIの欧文文字が小さくなってしまうことがあった。旧版 EBWin の方がまともに表示される。新版はバグっぽい動作をしたりクラッシュしたりと不安定だ。

この作者のソフトはドキュメントも不完全で どのように動作するのか細かい所がよくわからないことが多いが、他に選択肢もないのでそれらを使うしかない。掲示板は閉じてしまっているし、メールをしても無視されるだけ。
