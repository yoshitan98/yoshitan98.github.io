ハードディスク上のBDAVフォルダーをPCで再生する　2026年4月1日更新

　

10年前にパナソニックのレコーダーで録画しておいたBD-Rが劣化していて新品のレコーダーでも番組表示ができなくなっていた。BD-Rはノーブランドの安物を使ったわけではないのが・・・。

BDドライブ・メディアの命運が尽きようとしていることもあり、DRMを解除できる DVDFab Passkey を持っているので全部ハードディスクにコピーしておくことにした。

有償・無償問わずハードディスク上のBDAV対応のプレイヤーソフトがあるかどうかわからないのでバッチファイルで対応してしまうことにした。ダブルクリックで起動できる。フォルダー選択ダイアログボックスが出るのでBDAVフォルダーまたはその上位フォルダーを選択すれば番組リストが表示されるので番号を入力すると*.m2tsに関連付けしてあるPCのプレイヤーでそれが再生される（私はMPC-BEを使用している）。フォルダー中に複数のBDAVフォルダーが存在する時には最初に見つかったinfo.bdavファイルの番組リストが表示される。

レコーダー上でカット編集した動画がどう表示されるかはわからない。カット部分が静止画になったりするかもしれないが、レコーダーによって結果はち違う様子。

バッチファイル中で使用するコマンドは bdavinfo.exe と BusyBox for Windows の busybox.exe と nkf..exe で、いずれもパスを通しておいて下さい。

※蛇足ながら zip 中のファイルは *.bat.txt を *.bat とリネームして下さい。
